# Buyer Foresight – User Directory Dashboard

This is a small React app that shows a list of users. You can search, sort, and click on any user to see more details.

## Features

- View users in a table
- Search by name or email
- Sort by name or company
- Click a user to see full details
- Simple navigation between pages

## Tech used

- React
- Vite
- React Router
- Axios
